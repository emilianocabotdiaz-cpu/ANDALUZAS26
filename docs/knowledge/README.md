# Base de conocimiento de OpoCompi

Esta carpeta es el lugar donde iremos guardando el conocimiento propio de OpoCompi.

La primera version de la IA ya tiene instrucciones propias en:

```text
lib/opocompi-brain.ts
```

Mas adelante podemos conectar estos documentos al chat mediante busqueda semantica o una base de datos vectorial.

## Que meter aqui

- Temario revisado.
- Preguntas tipo test validadas.
- Explicaciones de respuestas correctas e incorrectas.
- Esquemas por bloque.
- Dudas frecuentes de opositores.
- Mensajes de motivacion y planificacion.

## Regla de calidad

No subir contenido dudoso como si fuera oficial. Todo lo normativo debe indicar fuente, fecha y si esta revisado.
